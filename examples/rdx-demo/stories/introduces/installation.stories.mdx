## @alife/rdx

状态管理框架, 通过控制数据流图的方式来控制组件的渲染，协同同步和异步事件流。

### 安装

`npm install @alife/rdx`

### 维护人员

@czw

### 特点

- 分解复杂交互
- 统一处理组件渲染前置依赖
- 统一维护模块加载状态
- 合适的取消前置依赖执行的实际
- 数据状态可视化工具

### 兼容性说明

表格用到了大量较新的浏览器 API，目前阶段只兼容较新版本的 Chrome/Firefox/Safari
