通过 getReferencedValueAtom函数，可以通过表单的字段路径的方式获取到表单的项的值atom。

---

```jsx
function getReferencedFormValueAtom<GPath extends TStringPathResolver<GSource>>(
  path: GPath
): RdxValueReference<TGetString<GSource, GPath>>;

```

**props**

- path 通过传入表单路径，获取表单值的atom。

---

import ExampleCustomizer from '../../../../src/components/ExampleCustomizer';
import * as Atom from '../../../../src/stories/rdxForm/api/formBuilder/getReferencedFormValueAtom';
import AtomSource from '!!raw-loader!../../../../src/stories/rdxForm/api/formBuilder/getReferencedFormValueAtom';
