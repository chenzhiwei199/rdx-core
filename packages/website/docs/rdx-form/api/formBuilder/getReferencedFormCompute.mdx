通过 getReferencedFormCompute 函数，可以通过表单的字段路径的方式获取到表单的项的compute状态。

---

```jsx
function getReferencedFormCompute<GPath extends TStringPathResolver<GSource>>(
  path: GPath
): RdxValueReference<TGetString<GSource, GPath>>;

```

**props**

- path 通过传入表单路径，获取表单项的compute。
