实现一个类型满足 `React.FC<IFormComponentProps<any>> | React.ComponentClass<IFormComponentProps<any>>`的组件，就可以通过 formBuilder 来
构建一个表单。

```jsx
export interface IFormComponentProps<T extends any> extends IBaseFormComponentProps {
  children?: React.ReactNode
   // 数组的操作方法, 使用该方法可以精确控制更新
   arrayHelper?: {
    mutators?: IArrayMutators
  }
  componentProps?:  T
}
```
